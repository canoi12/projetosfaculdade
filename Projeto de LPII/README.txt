Projeto de Linguagem de Programação II, usando linguagem de programação 
Java e a biblioteca de interface gráfica Swing. Foi pedido para usarmos 
duas estruturas vistas na disciplina de Estrutura de Dados Básica II, no 
caso Árvores Binárias(de busca, Rubro Negra, AVL, Trie[digital], 
Graduada), Heaps, e alguns conceitos de Orientação a Objetos 
(Polimorfismo, Herança, Interface, Classes Abstratas), classes de 
tratamento de erros.
